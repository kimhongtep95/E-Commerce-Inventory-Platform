# SearchBugs Web

React + TypeScript frontend for the SearchBugs bug tracking and project management platform.

## Stack

- React 18 + TypeScript
- Vite
- React Router
- TanStack Query
- Zustand
- Material UI
- React Hook Form
- Chart.js

## Run

```bash
npm install
npm run dev
```

Set `VITE_API_BASE_URL` to the SearchBugs API base URL before running.
